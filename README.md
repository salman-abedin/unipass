# unipass: Browser Independent Password Manager

# Dependencies

-  amenu/dmenu/rofi
-  gnupg
-  xsel
-  xdotool

## Installation

```sh
git clone https://github.com/salman-abedin/unipass.git && cd unipass && sudo make install
```

## Uninstallation

```sh
sudo make uninstall
```
